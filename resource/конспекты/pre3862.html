Шаг 1/3

К вступлению

В примере ниже, сообщение будет склеено из фрагментов строк и значений переменных.

package lab02; import java.util.Scanner; public class Task3862 { public static void main(String[] args) { Scanner scan = new Scanner(System.in); String x; x = scan.nextLine(); String y; y = scan.nextLine(); System.out.println("<point x=\"" + x + "\" y=\"" + y + "\">"); } }

В следующем примере, сообщение будет описано шаблоном. В нем с помощью символьной комбинации %s будут отмечены места для записи переменных. А после шаблона через запятую указаны переменные, значения которых нужно подставить на место этих комбинаций.

package lab02; import java.util.Scanner; public class Task3862 { public static void main(String[] args) { Scanner scan = new Scanner(System.in); String x; x = scan.nextLine(); String y; y = scan.nextLine(); System.out.printf("<point x=\"%s\" y=\"%s\">", x, y); } }

Переменная, указанная первой после шаблона была подставлена на место первой комбинации %s. Вторая переменная была подставлена вместо второй комбинации %s. Можно указывать множество переменных. Третья переменная будет подставлена вместо третьей комбинации и так далее.

На следующем шаге команда форматированного вывода будет применена для вывода чисел с нужной точностью.

Шаг 2/3

К шагу 1 К вступлению

По условию задачи нужно вывести значения π и e. Вы, конечно, можете найти в интернете значения этих математических констант и написать их вручную в код программы. Но намного проще применить встроенные в язык константы "Math.PI" и "Math.E". Если использовать их, то при опечатке, вы сразу же о ней узнаете, и программа не запустится. Опечататься в одной из цифр длинного числа намного легче, и при этом ошибка будет не заметна.

В прошлом примере была использована комбинация %s. С ее помощью выводятся строковые значения. При этом Math.PI и Math.E являются числами. Для них нужна комбинация %f

Выведите в консоль сообщение, содержащее значение этих констант.

package lab02; import java.util.Scanner; public class Task3862 { public static void main(String[] args) { System.out.printf("%f\n%f", Math.PI, Math.E); } }

В результате получится

3,14159265358979 2,71828182845905

Обратите внимание, что возле Math.PI и Math.E не нужны ". Эти значения постоянны, но технически реализованы как переменные с именами и запретом на изменение.

Теперь нужно ограничить эти значения четырьмя символами после запятой. Для этого нужно заменить %f на %.4f. Число "4" тут значит количество отображаемых символов после запятой. Более подробную справку о форматированном выводе можно посмотреть в документации (открыть в новой вкладке)

package lab02; import java.util.Scanner; public class Task3862 { public static void main(String[] args) { System.out.printf("%.4f\n%.4f", Math.PI, Math.E); } }

В результате получится

3,1416 2,7183

Вступление

В предыдущем разборе теории (открыть в новой вкладке) была рассмотрена команда для считывания значений, введенных пользователем в консоль.

Возможностей простой команды вывода в консоль может не хватить в двух случаях:

  • Вывод сложных сообщений, состоящих из множества фрагментов и значений переменных.
  • Вывод чисел с определенным количеством знаков после запятой.

В этих случаях полезно использовать команду форматированного вывода. Она состоит из технического оформления, шаблона сообщения и списка переменных. Самый простой пример оформления такой команды указан ниже. Будьте внимательны, в отличие от прошлой задачи для вывода используется не print, а printf

package lab02; import java.util.Scanner; public class Task3862 { public static void main(String[] args) { Scanner scan = new Scanner(System.in); String data; data = scan.nextLine(); System.out.printf("%s", data); } }
